# Llama2onCPU

# how to run 

### Steps 

clone the repository 

```bash
git clone https://github.com/DeepSadanand/Llama2onCPU.git
```

#### Step 2: 

create a virtual environment

```bash

conda create -n cpullama python=3.11 -y
```

```bash
conda activate cpullama
```
```bash
pip install -r requirements.txt
```
